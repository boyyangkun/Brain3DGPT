# Brain3DGPT
**3D Brain Image Anomaly Detection Using Anomaly-guided Large Vision-Language Models**
---

## 📌 Overview

Brain3DGPT is a unified anomaly-guided vision-language framework for 3D brain MRI analysis.  
It jointly supports:

- 🧠 Lesion localization  
- 🧠 Anomaly-aware reasoning  
- 🧠 Medical report generation  

The framework bridges volumetric medical imaging and large language models to enable interpretable and clinically meaningful multimodal understanding.

---

## 📊 Dataset

We introduce a new dataset:

### **MMBrainMRI-4K**

- 4,010 3D brain MRI volumes  
- Region-aware textual descriptions  
- Designed for joint evaluation of:
  - anomaly localization
  - multimodal reasoning
  - report generation
